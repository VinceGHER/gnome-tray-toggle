# Screenshots

Please add screenshots here for the README and GNOME Extensions website:

## Required Screenshots

1. **tray-visible.png** - Screenshot showing the tray with all icons visible
2. **tray-hidden.png** - Screenshot showing the tray with app icons hidden
3. **toggle-button.png** (optional) - Close-up of the toggle button

## Screenshot Guidelines

- Use PNG format
- Recommended resolution: 1920x1080 or similar
- Focus on the top panel area
- Show both states (visible/hidden)
- Clear, high-contrast images

## How to Take Screenshots

```bash
# Take a screenshot of a specific area
gnome-screenshot -a

# Or use the keyboard shortcut
# Shift + Print Screen
```

Then save the files in this directory with appropriate names.
